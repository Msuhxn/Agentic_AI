# Thesis Screenshot Guide

Use screenshots that show the prototype clearly without exposing private information.

## Recommended Screenshots

1. **Clean VS Code project structure**
   - Show the main Python files, `README.md`, `notebooks/`, and `outputs/`.

2. **Terminal command running successfully**
   - Show `python3 main.py` executing without errors.
   - Include the final line showing that evidence files were saved.

3. **Scenario 1 agentic output**
   - Show the terminal section `SCENARIO 1 - AGENTIC WORKFLOW OUTPUT`.
   - Include planner, writer, reviewer, and refiner stages if possible.

4. **Scenario 2 agentic refinement output**
   - Show the terminal section `SCENARIO 2 - AGENTIC WORKFLOW OUTPUT`.
   - Include reviewer and refiner stages.

5. **Evaluation results**
   - Show `outputs/evaluation_results.csv`.
   - The screenshot should show criteria, scores, and justifications.

6. **README or notebook preview for appendix**
   - Show the project purpose, scenarios, and evaluation criteria.

## Do Not Include

- `.env`
- API keys
- OpenAI billing dashboard
- credit balance
- personal account screenshots
- private emails, documents, or personal data
